# CPE241_NotiTask_DatabaseSystems
Frontend and Backend coding for NotiTask project

After open this folder by VS code
Step:
1) Open New terminal
   1.1) npm install
   1.2) cd server
   1.3) npm start
2) Then open another new terminal
   2.1) cd client
   2.2) npm i vite
   2.3) npm run dev
   You will get link to our project.
